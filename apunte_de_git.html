git

Git es un software de control de versiones, para instalarlo se debe descargar desde su página principal:
https://git-scm.com/download/win


configuración

para configurar git después de instalarlo debemos ir a la consola y poner los siguientes comandos:

git config --global user.name "el nombre que queremos mostrar"
para configurar un nombre para publicar. Revisar que el nombre se escribe entre comillas.
git config --global user.email mimail@servidor.com
para configurar el e-mail.

Comandos utiles

git init:
inicializa un repositorio.
debemos estar situados en la consola en una carpeta, y al poner este comando se creará un directorio nuevo llamado .git

git status:
para ver el estado del proyecto: los archivos modificados y los que están en el stage para commitear.

git add nombreDeArchivo
para agregar un archivo al stage.
git add .
para agregar todos los archivos al stage.

git commit -m "un comentario"
para commitear los archivos que tenemos en el stage. registramos los cambios del proyecto.

git checkout nombreDeArchivo:
para descartar los cambios realizados en ese archivo.
git checkout .
para descartar todos los cambios realizados de los archivos del proyecto. vuelve a como estaban en el último commit registrado.
